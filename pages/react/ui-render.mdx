## UI Rendering

## Components에서 변수 사용하기 

html 태그 안에서 변수를 사용하려면 중괄호 `{변수}`로 감싸주면 됩니다.

```ts filename="" showLineNumbers copy
const Footer = () => {
  const myName = "John Doe";
  return <footer>안녕 내 이름은 {myName}이야</footer>;
};

export default Footer;
```

### 3항 연산자 

- 3항 연산자를 사용해서 어떤 경우에 어떤 태그를 보여줄지 결정할 수 있음

```ts filename="" showLineNumbers copy
const Main = () => {
  const user = { name: "John Doe", login: true };
  return (
    <div>
      <h1>Main</h1>
      <p>{user.login ? <div> 로그아웃</div> : <div>로그인</div>}</p>
    </div>
  );
};

export default Main;
```

## Style 설정

### tag에 직접 넣는 방법

```ts filename="" showLineNumbers copy {6}
const Main = () => {
  const user = { name: "John Doe", login: false };
  return (
    <div>
      <h1>Main</h1>
      <p style={{ backgroundColor: "red", borderBottom: "1px solid blue" }}>
        {user.login ? <div> 로그아웃</div> : <div>로그인</div>}
      </p>
    </div>
  );
};

export default Main;
```

### 별도의 css 파일을 만들어서 적용하는 방법

```css filename="" copy showLineNumbers
.logout {
  background-color: red;
  border-bottom: 1px solid black;
}
```

- 이후 css 파일을 import해서 해당 태그의 class로 지정

```ts filename="" showLineNumbers copy {8}
const Main = () => {
  const user = { name: "John Doe", login: true };
  return (
    <div>
      <h1>Main</h1>
      <p>
        {user.login ? (
          <div className="logout"> 로그아웃</div>
        ) : (
          <div>로그인</div>
        )}
      </p>
    </div>
  );
};

export default Main;
```
