## Components

- Components를 선언하는 방법은 2가지가 있음

1. 함수형 

```ts filename="" showLineNumbers copy
function Header() {
  return (
    <header>
      <h1>Header</h1>
    </header>
  )
}
```

2. const 변수 

```ts filename="" showLineNumbers copy
const Header = () => {
  return (
    <header>
      <h1>Header</h1>
    </header>
  );
};
```

### 컴포넌트 사용하는 방법

```ts filename="" showLineNumbers copy
function App() {
  return (
    <>
      <Header />
      <h3>안녕! 리액트</h3>
    </>
  );
}
```
